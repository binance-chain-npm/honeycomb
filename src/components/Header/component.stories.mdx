# Header

### Header Item Props

| Name        | Type                     | Description                                           |
|-------------|--------------------------|-------------------------------------------------------|
| `target`    | `node`                   | The element inside the header item.                   |
| `children?` | `node` \| `undefined`    | If `node`, a dropdown will be rendered.               |
| `isStyled?` | `boolean` \| `undefined` | If `true`, no styling will be applied to the element. |
