# Header

### Header Item Props

| Name        | Type                  | Description                                                                                   |
|-------------|-----------------------|-----------------------------------------------------------------------------------------------|
| `target`    | `node` \| `string`    | If `node`, no styles will be applied. If `string`, the default header styles will be applied. |
| `children?` | `node` \| `undefined` | If `node`, a dropdown will be rendered.                                                       |
